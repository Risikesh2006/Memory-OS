## рҹӘһ What is Legacy OS?

Legacy OS is a personal memory operating system. Instead of scattering photos across cloud drives, videos across phones, and thoughts across random notes apps, Legacy OS gives every memory a home вҖ” timestamped, searchable, and woven into a single cinematic timeline of your life.

Think of it as **the director's cut of your own story.**

## рҹҗӣ Troubleshooting

<details>
<summary><strong>Database Connection Failed</strong></summary>

- Ensure MySQL is running
- Check credentials in `.env`
- Verify the database exists

</details>

<details>
<summary><strong>Port Already in Use</strong></summary>

- Change the port in `.env`
- Or kill the existing process occupying it

</details>

<details>
<summary><strong>CORS Errors</strong></summary>

- Verify the backend is running
- Check `NEXT_PUBLIC_API_URL` in your frontend `.env.local`

</details>

<details>
<summary><strong>Cloudinary Upload Fails</strong></summary>

- Verify your cloud name and API credentials
- Check that the upload preset exists and is unsigned/signed correctly

</details>
